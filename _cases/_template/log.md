# Case Log

> Append-only. Newest entries at top. Never rewrite history — if something was wrong, write a new entry correcting it.
>
> One line per entry. If you need more space, link to a separate dated file in this folder.

## Format

```
YYYY-MM-DD HH:MM — [specialist or initials] — what happened. → next: who/what.
```

Use `→ next:` to name the next handoff. This is what makes the log work as a handoff mechanism — every entry tells the next person what to do.

## Examples

```
2026-05-19 09:20 — 01_lead_qualifier (SM) — qualifier email drafted and sent. → next: await reply by 2026-05-22, then route to research.
2026-05-19 09:14 — 00_orchestrator (SM) — new contact via email referral from K. Patel. Case opened. → next: 01_lead_qualifier.
```
